# CODAP Templates

This project includes reusable templates for common DevOps work.

## Available templates

| Template | Path |
|---|---|
| Kubernetes deployment | `templates/kubernetes/deployment-template.yaml` |
| Kubernetes service | `templates/kubernetes/service-template.yaml` |
| Terraform tfvars | `templates/terraform/terraform.tfvars.template` |
| GitHub issue templates | `.github/ISSUE_TEMPLATE/` |
| Pull request template | `.github/pull_request_template.md` |
| Environment file | `templates/env.example` |

## How to use

Copy the template, replace placeholders, and commit the final file into the correct GitOps or Terraform folder.
